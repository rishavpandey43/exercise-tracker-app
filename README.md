## Technologies Used

- Front-End
  - ReactJS ( ^16.13.0 )
  - Bootstrap ( ^4.4 )
- Back-End
  - NodeJS ( ^v10.15.2 )
  - Express ( ^4.17.1 )

## Installation

- Front-End

  navigate to frontend folder

  `cd frontend`

  install npm dependencies

  `npm install`

  start the front-end-app

  `npm start`

- Back-End

  navigate to backend folder

  `cd backend`

  install npm dependencies

  `npm install`

  start the back-end-server

  `npm start`
